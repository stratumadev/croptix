## Setup

Make sure to install dependencies:

```bash
bun install
```

## Development (Manifest V2)
```bash
bun run dev:v2
```

## Development (Manifest V3)
```bash
bun run dev:v3
```

## Build Extension (Manifest V2)
```bash
bun run build:v2
```

## Build Extension (Manifest V3)
```bash
bun run build:v3
```
